# 🍻 Glass-per-hour FE: 기능 명세서

> '시간당 몇 잔?' 내 주량을 측정하고 친구들과 겨뤄보세요!

## 1. 개요 📜

**Glass-per-hour**는 사용자가 자신의 주량을 실시간으로 측정하고 다른 참가자들과 순위를 비교할 수 있는 웹 애플리케이션입니다. 측정 결과에 따라 재미있는 주량 레벨이 부여되어 즐거움을 더합니다.

## 2. 기술 스택 🛠️

-   **Framework**: React
-   **Language**: TypeScript
-   **Styling**: CSS

## 3. 화면 흐름 🚀

사용자는 아래와 같은 순서로 서비스를 이용하게 됩니다.

`🏁 시작` → `✍️ 이름 입력` → `📊 주량 측정` → `🏆 결과 확인`

## 4. 페이지별 기능 상세 📋

| 페이지 | 경로 | 주요 기능 |
| :--- | :--- | :--- |
| **🏁 시작** | `src/pages/StartPage.tsx` | <ul><li>서비스의 첫인상을 결정하는 소개 페이지</li><li>`시작하기` 버튼을 통해 이름 입력 페이지로 이동</li></ul> |
| **✍️ 이름 입력** | `src/pages/InputName.tsx` | <ul><li>랭킹에 사용할 사용자 이름(닉네임) 입력</li><li>입력 완료 시 주량 측정 페이지로 이동</li></ul> |
| **📊 주량 측정** | `src/pages/MeasurePage/` | <ul><li>**주종 선택**: 마신 술의 종류(소주, 맥주 등)를 선택</li><li>**잔 수 계산**: `+`, `-` 버튼으로 마신 잔 수를 실시간 기록</li><li>**실시간 랭킹**: 참가자들의 순위를 실시간으로 확인</li></ul> |
| **🏆 결과** | `src/pages/Result.tsx` | <ul><li>측정된 주량에 따른 최종 **레벨**과 **이미지** 표시</li><li>총 음주량 요약 정보 제공</li><li>`다시하기` 버튼으로 서비스 재시작</li></ul> |

## 5. 핵심 컴포넌트 🧩

주량 측정 페이지(`Measure.tsx`)는 아래의 핵심 컴포넌트들로 구성됩니다.

| 컴포넌트 | 경로 | 역할 |
| :--- | :--- | :--- |
| **DrinkList** | `DrinkList.tsx` | 선택 가능한 주종 아이템들을 표시 |
| **DrinkCounter** | `DrinkCounter.tsx` | 각 주종별로 마신 잔 수를 조절하는 카운터 UI |
| **RankingList** | `RankingList.tsx` | 실시간 주량 랭킹을 보여주는 리더보드 |

## 6. 서버 통신 📡

-   **API 정의**: `src/api/api.ts`
-   **주요 통신 내용**:
    -   사용자 이름 및 정보 등록
    -   마신 잔 수 실시간 업데이트
    -   참가자 랭킹 데이터 수신
    -   최종 주량 레벨 및 결과 요청
